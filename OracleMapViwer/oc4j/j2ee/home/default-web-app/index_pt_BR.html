Visão Geral

O Oracle Containers for J2EE 10g (10.1.3.5.0) é o principal componente de runtime do J2EE do Oracle Application Server.

Os recursos principais desta release são os seguintes:

  • J2EE 1.4 compatível com suporte adicional para a especificação EJB 3.0 final, incluindo o JPA 1.0 API
  • Recursos Extensivos de Web Services com suporte para JAX-RPC, WS-I, WS-Security, WS-Reliability
  • Solução de persistência relacionada a objeto líder da indústria com o Oracle TopLink
  • Gerenciamento baseado em padrões e suporte a disponibilização com o Application Server Control do Enterprise Manager
  • Desempenho e escalabilidade em nível corporativo

Verificação Rápida

 

Os exemplos não foram instalados porque muitos deles não atendem aos Padrões Oracle Secure Coding e podem não estar em conformidade com os padrões de codificação segura de sua empresa. Eles apresentarão vulnerabilidades de segurança significantes. No entanto, eles poderão ser instalados do diretório j2ee/home/default-web-app com o seguinte comando: "java -jar ExampleInstaller.jar install".
