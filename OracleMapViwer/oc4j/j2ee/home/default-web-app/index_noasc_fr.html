Aperçu

Oracle Containers for J2EE 10g (10.1.3.5.0) est le composant runtime J2EE principal d'Oracle Application Server.

Les apports-clés de cette version sont les suivants :

  • J2EE 1.4 compatible avec d'autres supports pour la dernière spécification EJB 3.0, y compris l'API JPA 1.0
  • Fonctionnalités Web Services étendues avec prise en charge de JAX-RPC, WS-I, WS-Security et WS-Reliability
  • Solution de persistance objet-relationel en tête du marché avec Oracle TopLink
  • Prise en charge d'un déploiement et d'une gestion basés sur les standards avec Enterprise Manager Application Server Control
  • Performances et évolutivité d'entreprise
Documentation

 

Le manuel Configuration and Administration Guide est inclus dans cette distribution.
Il est accessible à partir d'ici (Adobe Acrobat Reader est requis).

Le jeu de documentation complet est disponible sur Oracle Technology Network à l'adresse http://www.oracle.com/technology/documentation ; il contient :

  • Un guide de configuration et d'administration comportant des informations utiles sur la gestion du serveur
  • Un guide de déploiement comportant des informations utiles sur les opérations de déploiement
  • Des guides de développement aidant au développement d'applications J2EE faisant appel aux technologies JSP, Servlet et EJB
  • Des guides de développement comportant des informations utiles sur le développement et le déploiement de Web Services
  • Des guides sur la sécurité et les services décrivant les services standard disponibles, tels que JNDI et JMS, et l'infrastructure complète de sécurité

   
Gestion d'OC4J

 

OC4J est géré à partir d'un navigateur utilisant Application Server Control.

Cette instance est configurée pour être administrée à distance.
