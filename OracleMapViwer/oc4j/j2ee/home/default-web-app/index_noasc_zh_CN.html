概览

Oracle Containers for J2EE 10g (10.1.3.5.0) 是 Oracle Application Server 的核心 J2EE 运行时组件。

此发行版的关键价值在于：

  • 与 J2EE 1.4 兼容，并且提供了对最终 EJB 3.0 规范 (包括 JPA 1.0 API) 的支持
  • Web 服务功能可以广泛地支持 JAX-RPC、WS-I、WS-Security、WS-Reliability
  • 通过 Oracle TopLink 提供行业领先的对象关系持久性解决方案
  • 通过 Enterprise Manager Application Server Control 支持基于标准的管理和部署
  • 企业级性能和可扩展性
