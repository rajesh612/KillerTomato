Überblick

Oracle Containers for J2EE 10g (10.1.3.5.0) ist die zentrale J2EE-Laufzeitkomponente von Oracle Application Server.

Dieses Release bietet folgende Vorteile:

  • J2EE 1.4-Kompatibilität mit zusätzlicher Unterstützung für die endgültige EJB 3.0-Spezifikation einschließlich JPA 1.0 API
  • Umfangreiche Web Service-Funktionen mit Unterstützung für JAX-RPC, WS-I, WS-Security und WS-Reliability
  • Branchenweit führende objektrelationale Persistence-Lösung mit Oracle TopLink
  • Standardbasierte Verwaltung und Deployment-Unterstützung durch Enterprise Manager Application Server Control
  • Enterprise-Performance und -Skalierbarkeit

Schnelle Überprüfung

 

Es sind keine Beispiele installiert, weil viele von ihnen nicht den Oracle-Standards für eine sichere Codierung entsprechen und möglicherweise auch nicht die Standards Ihres Unternehmens für eine sichere Codierung erfüllen. Die Beispiele führen zu erheblichen Sicherheitslücken. Sie können jedoch aus dem Verzeichnis j2ee/home/default-web-app installiert werden. Verwenden Sie hierzu den folgenden Befehl: "java -jar ExampleInstaller.jar install".
