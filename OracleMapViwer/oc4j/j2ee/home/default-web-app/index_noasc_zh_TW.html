簡介

Oracle Containers for J2EE 10g (10.1.3.5.0) 是 Oracle Application Server 的核心 J2EE 程式實際執行元件。

此版本的主要功能：

  • J2EE 1.4 與 EJB 3.0 規格的其他支援相容 (包括 JPA 1.0 API)
  • 擴充「Web 服務」功能，此版本支援 JAX-RPC、WS-I、WS-Security 和 WS-Reliability
  • 業界首屈一指的 Oracle TopLink 物件-關聯解決方案
  • 「Enterprise Manager Application Server 控制」支援標準式的管理和建置
  • 企業效能和擴充性
