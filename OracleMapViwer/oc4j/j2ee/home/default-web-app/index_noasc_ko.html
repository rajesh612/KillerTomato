개요

Oracle Containers for J2EE 10g(10.1.3.5.0)는 Oracle Application Server의 핵심 J2EE 런타임 구성 요소입니다.

이 릴리스의 주요 장점은 다음과 같습니다.

  • JPA 1.0 API를 포함한 최종 EJB 3.0 사양에 대한 추가 지원으로 J2EE 1.4와 호환 가능
  • JAX-RPC, WS-I, WS-Security, WS-Reliability를 지원하는 다양한 웹 서비스 기능
  • Oracle TopLink를 통한 업계 최고의 객체 관계형 지속성 솔루션
  • Enterprise Manager Application Server Control을 통한 표준 기반 관리 및 배치 지원
  • 엔터프라이즈급 성능 및 확장성
