Visión General

Oracle Containers for J2EE 10g (10.1.3.5.0) es el componente principal de J2EE Runtime de Oracle Application Server.

Los valores clave de esta versión son:

  • J2EE 1.4 compatible con el soporte adicional para la especificación EJB 3.0 final, incluida la API JPA 1.0
  • Capacidades de servicios Web extensas con soporte para JAX-RPC, WS-I, WS-Security y WS-Reliability
  • Solución de persistencia relacionada con el objeto líder del sector con Oracle TopLink
  • Soporte con Enterprise Manager Application Server Control de gestión y despliegue basados en estándares
  • Escalabilidad y rendimiento de Enterprise
