Panoramica

Oracle Containers for J2EE 10g (10.1.3.5.0) è il componente runtime J2EE di base di Oracle Application Server.

Le caratteristiche principali di questa release sono:

  • compatibilità con J2EE 1.4 con supporto aggiuntivo per la specifica EJB 3.0 finale, inclusa l'interfaccia API di JPA 1.0;
  • funzioni avanzate di Web Services con supporto per JAX-RPC, WS-I, WS-Security, WS-Reliability;
  • soluzioni avanzate di persistenza relazionale a oggetti con Oracle TopLink;
  • supporto per gestione e distribuzione basate su standard con Enterprise Manager Application Server Control;
  • prestazioni e scalabilità Enterprise.
Documentazione

 

Il manuale Configuration and Administration Guide è incluso in questa distribuzione.
Per accedere a questo manuale, fare clic qui (è necessario disporre di Adobe Acrobat Reader).

La documentazione completa è disponibile sul sito di Oracle Technology Network http://www.oracle.com/technology/documentation e contiene quanto segue:

  • Configuration and Administration guide per le informazioni sulla gestione del server
  • Deployment guide per le informazioni sulle operazioni di distribuzione
  • Manuali di sviluppo per il supporto dello sviluppo di applicazioni J2EE che utilizzano le tecnologie JSP, Servlet e EJB
  • Manuali di sviluppo per il supporto dello sviluppo e della distribuzione di Web Services
  • guide alla sicurezza e ai servizi in cui sono descritti i servizi standard disponibili, quali JNDI e JMS, e l'infrastruttura completa di sicurezza.
