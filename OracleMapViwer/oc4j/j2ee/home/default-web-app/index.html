Overview

Oracle Containers for J2EE 10g (10.1.3.5.0) is the core J2EE runtime component of Oracle Application Server.

The key values of this release are:

  • J2EE 1.4 compatible with additional support for the final EJB 3.0 specification, including JPA 1.0 API
  • Extensive Web Services capabilties with support for JAX-RPC, WS-I, WS-Security, WS-Reliability
  • Industry leading Object-Relational persistence solution with Oracle TopLink
  • Standards based management and deployment support with Enterprise Manager Application Server Control
  • Enterprise performance and scalability
Documentation

 

The Configuration and Administration Guide is included within this distribution.
It can be be accessed from here (requires Adobe Acrobat reader).

The full documentation set is available from Oracle Technology Network at http://www.oracle.com/technology/documentation and contains:

  • Configuration and Administration guide to help with management of the server
  • Deployment guide to help with deployment operations
  • Development guides to support the development of J2EE applications using JSP, Servlet, and EJB technologies
  • Development guides to support the development and deployment of Web Services
  • Security and Services guides which describe the standard services available such as JNDI and JMS and the comprehensive security infrastructure
