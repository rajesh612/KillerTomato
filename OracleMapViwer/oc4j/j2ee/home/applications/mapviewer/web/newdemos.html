10.1.2 Demos

On-line JSP Demos (since 10.1.2)

These are the demos that come with your MapViewer 10.1.2 deployment. Note that to try the Region Management and Topology demos, you must download and install the new demo data set (mapviewer_1012_demos.zip). The new demo dataset can be downloaded from OTN MapViewer site, under the Software section. All the demos require that you use a 10g database or later version.

Also note that since all the demos are written in JSP, you can always check out the source code by going to the directory where the JSP files are located, which is $MAPVIEWER_HOME/web/demo directory.

  • terrinit.jsp A powerful & interactive Region Management demo using MapViewer's SVG capabilities.
  • topology.jsp A simple JSP demo that you can use to view Oracle Spatial 10g Topology data.
  • network.jsp A simple JSP demo that you can use to view any Oracle Spatial Network Data Model data.
  • georaster.jsp A simple JSP demo that you can use to view any Oracle Spatial GeoRaster data.
