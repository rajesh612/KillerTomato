If you are using OC4J or Oracle Application Server version 10.1.3 or higher, the login user name should be 'oc4jadmin'; on earlier versions you may try the user name 'admin'.

The password is the same as that of the OC4J's administrative account, which you setup when starting OC4J for the first time, or through the Oracle AS' EM web site as instructed in MapViewer's User Guide.
