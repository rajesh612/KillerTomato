 49    @class MVMapDecoration defines a stationary map decoration feature on the map. 
 50    The map decoration object is not moved when the map is dragged or 
 51    recentered. Its size 
 52    is not changed when the map is zoomed in or out. The MVMapDecoration is 
 53    displayed above all base map and FOI layers. <br><br>
 54    The positioning of the MVMapDecoration object is 
 55    specified by its position relative to the upper-left corner from the map. 
 56    The relative position is not specified in pixels, instead it is specified 
 57    in the percentage relative to the map window size. <br><br>
 58    The horizontal position 
 59    is specified by parameter left, which should be a float number in the range of 0-1. 
 60    The absolute offset of the MVMapDecoration object center from the left map 
 61    window border in pixels is left*map_window_width. <br><br>
 62    The vertical position is 
 63    specified by parameter top, which should be a float number in the range of 0-1. 
 64    The absolute offset of the MVMapDecoration object center from the top map 
 65    window border in pixels is top*map_window_height.

242   This method sets the position of the map decoration object relative to its position specified by the left and 
243   top parameter in the constructor.
244   @param {float} left left specifies the horizontal 
245                  position of the map decoration object relative to the left border of 
246                  the map window.
247   @param {float} top top specifies the vertical position of the map decoration  
248                  object relative to the top border of the map window.
249   @param {int} xOffset xOffset is an integer that specifies the horizontal offset in pixels
250   @param {int} yOffset yOffset is an integer that specifies the vertical offset in pixels
251   @returns None
252   @type void
253  */
254  MVMapDecoration.prototype.setPosition = function(left, top, xOffset,yOffset)
255  {
256    this.shouldAdjustCopyright = false ;
257    if(!xOffset)
258      xOffset = 0 ;
259    if(!yOffset)
260      yOffset = 0 ;
261    this.xOffset = xOffset;
262    this.yOffset = yOffset;
263    
264    if(typeof(left)=="number")
265     this.left = left;
266    else
267    {
268     this.left = 1 ;
269     if(!xOffset && this.width)
270       xOffset = -this.width ;
271    }
272    if(typeof(top)=="number")
273     this.top = top;
274    else
275    {
276     this.top = 1 ;
277     if(!yOffset && this.height)
278       yOffset = -this.height ;
279    }
280    
281    if(!this.parent)
282      return ;
283      
284    if(this.collapsible)
285    {
286      if(!this.collapsed)
287        this.initPosition(this.contentdiv);
288    }
289    else
290      this.initPosition(this.fixedDiv);
291  }

306    This methods sets whether the map decoration object is collapsible. 
307    
308    <p>If it is 
309    collapsible, the map decoration object can be minimized and displayed as a 
310    minimized icon by clicking the minimize button. It can be then restored 
311    to its original size by clicking the minimized icon. Currently the position 
312    parameters specified in the MVMapDecoration constructor do not have any 
313    effect on a collapsible map decoration, which is always placed at the lower 
314    right corner of the map.<p>
315    
316    <p>If the map decoration object is collapsible, this method can set its visual style.
317    The style parameter is an object that specifies various aspects, such as the map decoration window border,
318    title bar, minimize button and restore button. The object has the following attributes:
319    <UL>
320      <LI>borderStyle - The CSS style string for map decoration window borders.
321      <LI>titleBgImg - The URL of the title bar background image. 
322      <LI>titleBgImgMO - The URL of the title bar background image displayed when the mouse moves over it.
323      <LI>tileBgImgMD - The URL of the title bar background image displayed when the mouse is pressed down on the title bar button. 
324      <LI>minBtnImg - The URL of the minimize button image.
325      <LI>minBtnImgMO - The URL of the minimize button image displayed when the mouse moves over it.
326      <LI>minBtnImgMD - The URL of the minimize button image displayed when the mouse is pressed down on it. 
327      <LI>resBtnImg - The URL of the restore button.
328      <LI>resBtnImgMO - The URL of the restore button displayed when the mouse moves over it.
329      <LI>resBtnImgMD - The URL of the restore button displayed when the mouse is pressed down on it.
330    </UL>
331    <p>Here is an example.</p>
332    <pre>
333    var style = {
334      borderStyle :"1px solid #636661",
335      titleBgImg  :"/images/title_bg.png", 
336      titleBgImgMO:"/images/title_bg_over.png", 
337      tileBgImgMD :"/images/title_bg_over.png",
338      minBtnImg   :"/images/button_minimize.gif",
