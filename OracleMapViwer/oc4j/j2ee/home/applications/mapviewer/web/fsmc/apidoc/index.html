Class Index

MVArrayParameter

MVArrayParameter defines an array type binding parameter for a theme-based FOI layer that is based on a templated predefined theme.

MVBarChartStyle

MVBarChartStyle represents the client side Bar Chart style.

MVBarInfo

MVBarInfo this class represents the definition of a single bar in a Bar chart.

MVBingTileLayer

MVBingTileLayer defines a map tile layer that displays Microsoft Bing map tiles using Bing Maps Javascript API.

MVBucketSeries

MVBucketSeries defines a container for a series of buckets used by MapViewer thematic mapping.

MVBucketStyle

MVBucketStyle defines an ADVANCED type style that is defined through a series of buckets with their associated values and styles.

MVButtonGroup

The MVButtonGroup class defines a group of buttons.

MVCircleTool

MVCircleTool defines a circle tool that can be used to interactively draw a circle on the map by holding down and dragging the mouse.

MVCollectionBucket

MVCollectionBucket defines a bucket containing a list of items (which can be either string or numeric literals).

MVCustomMapTileLayer

MVCustomMapTileLayer defines a custom map tile layer, which assembles and displays map image tiles served directly by a web based map tile server, which accepts HTTP GET map tile requests.

MVDistanceTool

MVDistance defines a distance tool instance with which the user can calculate the straight-line distance between the points generated by clicking and moving the mouse on the map repeatedly.

MVEvent

MVEvent defines all Oracle Maps client side event types, which should be used when specifying the type of event associated with the event listener function.

MVFOI

MVFOI defines a feature of interest (FOI) object.

MVFOIGroup

MVFOIGroup defines a group of user defined FOI(MVFOI) objects.

MVGoogleTileLayer

MVGoogleTileLayer defines a map tile layer that displays Google Maps using Google Maps Javascript API.

MVInfoWindowTab

MVInfoWindowTab creates an information window tab.

MVMapDecoration

MVMapDecoration defines a stationary map decoration feature on the map.

MVMapTileLayer

MVMapTileLayer defines a map tile layer displayed on the client side.

MVMapView

MVMapView is the main entry class for all map operations inside the web browser.

MVNavigationPanel

MVNavigationPanel defines a map navigation panel that is to be displayed inside a map decoration(MVMapDecoration).

MVNSDP

MVNSDP provides info about a Non Spatial Data Provider for a theme based FOI layer.

MVNumericRangedBucket

MVNumericRangedBucket defines a bucket containing a numeric range.

MVOverviewMap

MVOverviewMap defines an overview map that displays the miniature overview of the main map as a small rectangle inside itself.

MVPieChartStyle

MVPieChartStyle represents a client side Pie chart style.

MVPieSlice

MVPieSlice represents the definition of a single pie slice.

MVRectangleTool

MVRectangleTool defines a rectangle tool that can be used to draw a "rubber-band" style rectangle on the map, by holding down and dragging the mouse.

MVRedlineTool

MVRedlineTool defines a redline tool instance with which the user can draw a polyline by clicking and moving the mouse on the map repeatedly.

MVScaleBar

MVScaleBar defines a scale bar object that is to be displayed inside a map decoration(MVMapDecoration) object.

MVSdoGeometry

MVSdoGeometry defines a 2-D geometry object that is similar to the SQL SDO_GEOMETRY type in Oracle Spatial or Locator.

MVSdoPointType

MVSdoPointType defines a point object that can be used to construct a point type MVSdoGeometry object.

MVStyleColor

MVStyleColor represents a client side constructed MapViewer COLOR style.

MVStyleMarker

MVStyleMarker lets you create a MapViewer MARKER style instance at the client side.

MVThemeBasedFOI

MVThemeBasedFOI defines a theme based FOI layer.

MVToolBar

MVToolBar defines a draggable toolbar that acts as a placeholder of multiple control buttons.

MVToolButton

MVToolButton defines a clickable tool button displayed in a MVToolBar object.

MVXMLStyle

MVXMLStyle lets you create a MapViewer style using the valid XML style definition at the client side.
